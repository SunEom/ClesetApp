## 프로젝트 개요

패션에 대한 관심이 증가하고 유행이 빠르게 변화하면서 많은 사람들이 의류에 소비하는 비용이 증가하고 있다. 이로 인해 옷장은 옷으로 가득 차고, 자신이 소유한 옷을 기억하지 못하는 경우가 많다. 이러한 문제를 해결하기 위해 의류 관리 시스템이 필요해졌으며, 이를 통해 사용자는 옷을 카테고리별로 분류, 저장, 수정, 삭제, 조회, 좋아요 기능을 활용해 체계적으로 관리할 수 있다. 프로그램은 사용자의 의상 데이터를 기반으로 계절에 맞는 옷이나 부족한 카테고리의 의상을 제안해주어, 적절한 옷차림을 참고하거나 중복 구매를 줄이는 데 도움을 준다.

또한, 이 프로그램은 사용자들에게 패션 커뮤니티를 제공하여 자신의 스타일을 공유하고 다른 사람들의 스타일을 참고할 수 있다. 의류 보관 방법이나 코디 팁 등 유용한 정보를 나누고 얻을 수 있다. 이를 통해 사용자는 소유한 의상을 물리적 공간의 제약을 넘어 더 효율적으로 관리할 수 있게 된다.

## 기술 스택

- Swift, SwiftUI
- Combine, MVVM
- Firebase Auth
- Animation
- KingFisher
- Restful API
